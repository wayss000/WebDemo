# WebDemo

## 项目说明

&emsp;&emsp;一个基于SSM的Web脚手架工程，在这里我会从零开始，一步步引入模板语言，引入MySQL，引入Redis，引入Elasticsearch，引入MQ，同时模块分层，并且会将我平时写的不错的可以通用的类，或者不错的设计放进来。

&emsp;&emsp;在我每完成一个功能我会打一个tag，对于仅需要部分功能的朋友来说，可以根据tag选择自己需要的版本。
